# Arvore
Arvore Binaria feita em java, codigo humano, humildade, sem llms. Raro hoje em dia ein

## Como executar:
```bash
git clone https://github.com/JoaoVLima/Arvore.git
```
```bash
cd Arvore
```
```bash
mvn compile exec:java
```
